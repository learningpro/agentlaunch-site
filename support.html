AgentLaunch 技术支持

AgentLaunch 是一款 macOS 工具，帮助你一键启动 Claude Code、Codex 等 AI 编程助手，支持多配置管理、自定义环境变量和多终端选择。

常见问题

Q: AgentLaunch 支持哪些终端？

支持 macOS 自带的 Terminal.app 和 iTerm2。

Q: AgentLaunch 支持哪些 CLI 工具？

内置支持 Claude Code 和 Codex，同时支持自定义 CLI 命令。

Q: 需要联网吗？

AgentLaunch 本身不需要联网。它只是帮你配置和启动终端会话，CLI 工具本身可能需要网络连接来访问 API 服务。

Q: 如何设置 API Key？

在配置编辑器中填写 API Key 字段，AgentLaunch 会在启动终端时自动设置对应的环境变量。API Key 仅存储在本地设备上。

Q: 支持哪些 macOS 版本？

需要 macOS 14.0 (Sonoma) 或更高版本。

Q: 如何导入/导出配置？

在主界面点击导出按钮，可以导出全部或单个配置为 JSON 文件。导入时选择对应的 JSON 文件即可。
